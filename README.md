# 🕌 Prayer Times & Qibla App

A modern Flutter application that provides accurate prayer times, a Qibla compass, location-based calculations, offline storage, and daily prayer notifications.

Designed with an offline-first approach to ensure a fast and reliable user experience.

---

# ✨ Features

- 🕌 Accurate Prayer Times
- 🧭 Qibla Compass
- 📍 Automatic Location Detection
- 🌎 Reverse Geocoding
- 🔔 Daily Prayer Notifications
- ⏰ Alarm Scheduling
- 💾 Offline Data Storage
- ⚡ Fast Local Performance
- 📱 Material Design Interface
